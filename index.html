How ProgramMetrics Works

ProgramMetrics starts with a clear file-to-dashboard workflow. Upload a supported file, review the dashboard, adjust setup while the session is active, and download the outputs needed for reporting or follow-up analysis.

  • Upload a standard Excel or CSV file
  • Choose the dashboard type and header row
  • Preview KPIs, charts, tables, filters, and data quality checks
  • Download a report summary, chart exports, and cleaned data
  • Use Tier 2 or Tier 3 when the project needs configuration or custom systems

Common Questions

Details customers usually want before choosing a dashboard or reporting path.

What file types does Studio support?

Tier 1 Studio is designed for standard Excel and CSV files.

Can customers upload more than once?

Yes. Paid Studio access includes repeated uploads during the active browser session so customers can adjust the file and header row before final downloads.

When should a customer use Tier 2?

Use Tier 2 when the dashboard needs selected branding, custom KPIs, calculated metrics, report formatting, or an interactive HTML report.

Does Tier 1 store uploaded files?

Tier 1 Studio files are processed in the browser session. Review the Privacy Policy for current data handling details.
