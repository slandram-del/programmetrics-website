What Tier 2 Covers

Tier 2 is for customization that can be captured in a structured configuration. It is a fit when the standard Studio dashboard works, but the organization needs more control over the look, metrics, charts, filters, downloads, or report package.

Branding

Organization name, logo placement notes, color preferences, labels, section names, and report tone.

Dashboard Logic

Priority KPIs, calculated fields, chart types, filters, groupings, tables, and dashboard layout.

Downloads

PDF summaries, PNG chart exports, cleaned CSV or Excel files, and optional interactive HTML reports.
