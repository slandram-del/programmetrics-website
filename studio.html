Studio Questions

The basics before choosing a Studio tier.

What does Tier 1 include?

Tier 1 includes upload, dashboard generation, browser preview, and standard report downloads using standard templates.

Can Tier 1 be customized?

No. Tier 1 uses standard templates only. Customization starts at Tier 2.

What file types can be used?

Studio is designed for Excel and CSV files as the starting point.

Can Studio become a custom app?

Yes. Tier 3 and Tier 4 support custom applications, automation, advanced analytics, integrations, and specialized reporting systems.

Want to use ProgramMetrics Studio?

Ask about Tier 1 if you want a standard upload-to-dashboard workflow, or describe what you need customized if you are ready for Tier 2 or above.
