Studio Questions

Clear boundaries for the self-service tier.

Do users need ProgramMetrics to run Tier 1?

No. Tier 1 is self-service. Users upload their file, generate the dashboard, and download outputs in the browser.

Are files uploaded to a server?

No. This version processes files locally in the user's browser for standard dashboard generation.

Can Tier 1 be customized?

No. Tier 1 uses standard templates. Customization starts at Tier 2.

What file types work?

CSV works directly. Excel files are supported in modern browsers through the included spreadsheet reader.

Need customization?

Use Tier 1 yourself when standard templates are enough. Contact ProgramMetrics when you need Tier 2 customization or a custom application.
